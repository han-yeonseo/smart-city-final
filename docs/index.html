06 / 핵심 발견

성공과 실패의 결정적 차이

데이터 분석을 통해 도출한 판교 성공·청라 실패 요인

1.8×
교통 접근성 격차
판교역(신분당선+경강선 환승)은 30분 등시간권 내 91개 역에 도달 가능. 청라역(공항철도 단일노선)은 51개 역에 그침. 노동시장 규모에서 판교가 청라보다 1.8~2.1배 우위.
7.5×
업무기능 집적도 격차
판교 업무시설 비율 42% vs 청라 5.6%. 청라국제업무지구 내 업무시설 건축물은 단 5개(연면적 398,171㎡)에 불과. 2005년 개발계획 승인 후 21년째 업무지구로서 기능 미완성.
2.0×
개발 밀도 격차
판교 평균 용적률 221.9% vs 청라 114.3%. 판교는 고밀 복합개발로 업무·주거·상업 기능이 공존하는 자족도시 구조를 실현. 토지이용 혼합도(엔트로피)도 0.362 vs 0.288로 판교 우위.
